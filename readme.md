# 🌍 HORIZON — Country Explorer

HORIZON is a modern and interactive **Country Explorer web app** that allows users to search, filter, and explore detailed information about countries around the world.

---

## 🚀 Features

- 🔍 **Search Countries**
  - Search any country by name

- 🌎 **Filter by Region**
  - Africa, Americas, Asia, Europe, Oceania
  
- 📊 **Country Details View**
  - Capital
  - Population
  - Area
  - Region & Subregion
  - Timezone
  - Flag

---

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (Vanilla JS)
- REST API: [REST Countries API](https://restcountries.com/)

---


---

## ⚙️ How It Works

1. User enters a country name or selects a region
2. App fetches data from REST Countries API
3. Country cards are displayed dynamically
4. Clicking a card shows detailed information
5. UI switches between:
   - Country list view
   - Country detail view
